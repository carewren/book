#Portfolio site

##Tools
###Workflow
* **Markdown Editor** [MacDown] (https://macdown.uranusjr.com/)
* **Text Editor** [Atom] (https://atom.io/)
* **FTP Client** [Transmit] (https://panic.com/transmit/)
* **Server** [MAMP] (https://www.mamp.info/en/)

###Build
* **CMS** [Grav] (https://learn.getgrav.org/)
